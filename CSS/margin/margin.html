你應該知道的:

  1. 想避免區塊元素從左到右撐滿容器，要記得設定區塊元素的 width 屬性。 然後你可以設定左右外邊距（margin-left 與 margin-right）為 aoto 來使其 水平居中。元素在顯示的時候，只會顯示到你所指定的寬度，然後剩餘的寬度平均 的散落在左右兩邊的邊距上。

  2. 得先了解Margin collapse。不然在用block的垂直排版前，會出現一堆迷之間距。 簡單來說就是 block 元素們之間(兩個皆是 block)的 margin-top 和margin-bottom 赤裸裸相遇沒有 border 或 padding 隔開時 ，會產生重疊合成單一的 margin-top 和margin-bottom (其值會選擇最大值)。Margin-collapse 水平方向無此情況。

  3. 會發生 Margin collapse 的基本情況如下

    1. 相鄰的兄弟姊妹元素
    2. 父層和自己的第一個/最後一個子元素
    3. 解決方法可以使用 padding 解決，不要使用 margin
